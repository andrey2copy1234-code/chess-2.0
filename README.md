# chess-2.0
проект шахмат. с ботом и многим другим - 2 версия. имеет сохранения и полностью совместим с chess 1.0 и баги тоже имеет
## Запуск
Скачайте с releases .zip файл и запустите файл chess.exe
## Управление
на кнопку b запускается бот
на кнопку s включается/выключается переворот экрана
на кнопку a запускается анализ ввиде графиков где показывается как вы сыграли партию
## Бот
Работает на minimax с альфа-бета отсечением и оценочной функции ввиде цена всех фигур моих-цена всех фигур врага. оценка происходит прямо во время расчёта (на всех глубинах) а не когда уже depth=1.
## Шахматный движок
Использует int8_t[8][8] для хранения фигур где +фигуры это белые -фигуры это чёрные а то насколько +n фигура или -n фигура это её цена (не повторяется). вот таблица цен:
 Имя фигуры | abs(f) | Индификатор в файле | Именование в коде |
| :--- | :---: | :---: | ---: |
| Пустота | 0 | - | `figureType::Empty` |
| Пешка | 1 | `пешка` | `figureType::Pawn` |
| Конь | 3 | `конь` | `figureType::Knight` |
| Слон | 4 | `слон` | `figureType::Bishop` |
| Ладья | 4 | `лодья` | `figureType::Rook` |
| Ферзь | 9 | `ферзь` | `figureType::Queen` |
| Король | 40 | `король` | `figureType::King` |
Как видно король тут 40 но фактически он не используется в оценке в случае взятия короля считается счёт по формуле 900000+100*depth+score где depth это от максимальной глубины просчёта бота (сейчас 8) до 1 где 1 это конец.
